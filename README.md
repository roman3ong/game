# 有选择回复 · 评论系统

基于截图中小红书风格的社交评论界面，实现了「有选择回复」功能。

## 功能

- **单条回复**：点击任意评论的"回复"按钮，展开内联输入框，直接回复该条评论。
- **有选择回复（批量）**：
  1. 点击右上角「选择回复」按钮进入选择模式。
  2. 点击勾选一条或多条评论（支持全选）。
  3. 点击「回复所选」弹出底部弹窗，输入一次内容后同时发送给所有选中用户。
- **发布新评论**：底部输入框直接发表顶层评论。
- **点赞**：点击爱心按钮切换点赞状态。

## 运行

直接用浏览器打开 `index.html` 即可，无需任何构建工具或依赖。
